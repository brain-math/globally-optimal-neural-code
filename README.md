# globally-optimal-neural-code
Code accompanying the paper "The geometry of globally optimal neural representations" (2026) by Gengshuo John Tian and Brent Doiron.

Sim.m contains code for numerical simulations (Figs. 1, 3, 6, 7, S1).

Analysis.m contains code for data analysis (Figs 4, 5). Requires a head direction cell dataset from Duszkiewicz et al. (2024), which can be downloaded via https://doi.org/10.6084/m9.figshare.24921252
This link also includes the MATLAB analysis code used in the original paper. Our analysis uses some of the dependencies in the code, in particular the TSToolbox (https://github.com/PeyracheLab/TStoolbox).
